# MacOs or Linux

```bash
$ python3 -m pip install --user virtualenv

$ python3 -m venv env
```

# Windows

```bash
$ py -m pip install --user virtualenv

$ py -m venv env
```

## Running the app

```bash
# Run this command to run automated tests
$ pytest test.py
```
